# Employee Management with Attendance System

## Project Description

Employee Management with Attendance System is a JavaFX desktop application developed using Java 21 and Maven.

The system allows users to:

* Login to the system
* Manage employee information
* Track employee attendance
* Generate reports
* Store data using local file storage
* View attendance statistics through the dashboard

---

## Features

### Login System

* User login interface
* Logout functionality
* Navigation between screens

### Dashboard

* Employee count
* Present employee count
* Late employee count
* Absent employee count

### Employee Management

* Add Employee
* Update Employee
* Delete Employee
* Search Employee
* Employee TableView

### Attendance Management

* Check In
* Check Out
* Mark Absent
* Attendance Search
* Attendance Percentage Calculation

### Reports

* Employee Report
* Attendance Report

### Data Persistence

* Save Employees to File
* Load Employees from File
* Save Attendance to File
* Load Attendance from File
